Notes - Git

Commands

  • config: used to set up your username and email
  • clone: download a remote repository to your computer (becomes a local repository you can work on)
  • status: checks the working directory
  • pull: synchronise you local repository with the remote one
  • add -A: move all files to the staging area
  • commit -m "message": commit your changes to the local repository
  • push: move your changes to the remote repository

Git ignore

This is a kind of file that allows you to specify what files git should ignore. You probably dont want to keep track of files like compiled code or files created by a program during testing in your repositories and git ignore allows you to do this.
